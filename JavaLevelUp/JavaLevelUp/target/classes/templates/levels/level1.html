LEVEL 1

Theoretical part

Variables are used to store data in the program. A variable represents a named memory area that stores a value of a certain type. Each variable has a type, name, and value. The type defines what information a variable or a range of valid values can store.

Variables are declared as follows:
data_type variable_name;
For example, let's define a variable that will be called x and will be of type int:
int x;


One of the main features of Java is that this language is strongly typed. This means that each variable and constant represents a certain type and this type is strictly defined. The data type defines the range of values that a variable or constant can store.

So, let's look at the system of built-in basic data types that is used to create variables in Java. And it is represented by the following types.

boolean: stores the value true or false
boolean isActive = false;
boolean isAlive = true;
byte: stores an integer from -128 to 127 and takes 1 byte
byte a = 3;
byte b = 8;
short: stores an integer from -32768 to 32767 and takes 2 bytes
short a = 3;
short b = 8;
int: stores an integer from -2147483648 to 2147483647 and occupies 4 bytes
int a = 4;
int b = 9;
long: stores an integer from -9 223 372 036 854 775 808 to 9 223 372 036 854 775 807 and occupies 8 bytes
long a = 5;
long b = 10;
double: stores a floating-point number from ±4.9*10-324 to ±1.8*10308 and occupies 8 bytes
double x = 8.5;
double y = 2.7;
A dot is used as the separator of the integer and fractional parts in fractional literals.
float: stores a floating-point number from -3.4*1038 to 3.4*1038 and takes up 4 bytes
float x = 8.5F;
float y = 2.7F;
char: stores a single UTF-16 encoded character and occupies 2 bytes, so the range of stored values is from 0 to 65535.
